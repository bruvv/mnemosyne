# Configuration

Mnemosyne is configured entirely through environment variables. No config files, no YAML, no JSON.

---

| Variable | Default | Description |
|----------|---------|-------------|
| `MNEMOSYNE_DB_PATH` | `~/.mnemosyne/mnemosyne.db` | Path to the SQLite database |
| `MNEMOSYNE_EMBEDDING_MODEL` | `all-MiniLM-L6-v2` | SentenceTransformer model for vector embeddings |
| `MNEMOSYNE_EMBEDDING_DEVICE` | `cpu` | Device for embeddings: cpu, cuda, mps |
| `MNEMOSYNE_BEAM_ENABLED` | `true` | Enable BEAM (Budget Elastic Adaptive Memory) tiering |
| `MNEMOSYNE_BEAM_CPU_THRESHOLD` | `5000` | Max working memories before cold tier promotion |
| `MNEMOSYNE_BEAM_COLD_THRESHOLD` | `10000` | Max cold memories before frozen tier promotion |
| `MNEMOSYNE_MCP_TOKEN` | *(required)* | Bearer token for MCP server auth (required for remote deployment) |
| `MNEMOSYNE_MCP_HOST` | `0.0.0.0` | MCP server bind address |
| `MNEMOSYNE_MCP_PORT` | `4321` | MCP server port |
